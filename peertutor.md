# PeerTutor Development & Launch Log

This log tracks the complete journey of PeerTutor — from the first lines of code
to a live, school‑ready platform. Early entries are approximate; exact dates can be
verified via GitHub commit history.

---

## 2026-02
- Began experimenting with AI tools (ChatGPT, DeepSeek) to generate code.
- Conceived the idea of a peer‑tutoring platform for DBS students, since I was met with the same problem I believe a lot of students in my school have.

## 2026-03
- Built the first prototype of PeerTutor entirely with AI‑generated code.
- Realised I was copy‑pasting without understanding; felt like a fraud.
- Wrote a public confession on GitHub, admitting my mistake and promising to rebuild honestly.
- Started over from scratch.

## 2026-04
- Rebuilt PeerTutor using React, Firebase (Auth, Firestore, Hosting), and CSS.
- Studied each concept thoroughly before committing code.
- Implemented core features: Google sign‑in (restricted to @dbs.edu.hk), user profiles, request feed, offer/help flow.
- Added points system to incentivise participation (1 point to post, earn points by helping).

## 2026-05
- Deployed the first live version at peertutor-dbs.web.app.
- Added teacher dashboard for oversight (view all requests, matches, reports).
- Added report/block functionality for safety.
- Shared the platform with Ms. Wong for initial feedback.

## 2026-06
- Met with Ms. Wong; she advised on safety and teacher oversight.
- Added urgent flag feature: 1 free use per week (3 per exam month) to boost requests.
- Added leaderboard and profile enhancements.
- Fixed critical bugs (e.g., report button redirecting to homepage).
- Began planning a school‑wide pilot launch.

## 2026-07
- **12 July:** Added AI‑powered content moderation (swear filter) based on Ms. Wong’s feedback.
- **12 July:** Updated app name to “peertutor” with subtitle “for diocesans.”
- **13 July:** Met Ms. Wong after school; she confirmed no formal teacher sponsor is required for launch and encouraged me to proceed with the built‑in safety features.
- **16 July:** Finalised launch plan: target pilot with 10–20 students in September.
- **17 July:** Set up this documentation repository to track future user numbers, testimonials, and iterations.
